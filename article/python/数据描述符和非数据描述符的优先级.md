定义了`__set__`或`__delete__`的描述符是数据描述符，优先级高，在查找_dict__之前调用同名的数据描述符。

只定义了`__get__`的是普通描述符，先查找__dict__之后才会查找它。


|优先级|机制名称|核心性质|说明|
|---|---|---|---|
| 1 (最高) | `__getattribute__` | 唯一入口 | 只要你访问 obj.x，它就一定会最先被调用 |
| 2 | 数据描述符 | 类属性 | 同时定义了 `__get__` 和 `__set__` 的类。如 @property |
| 3 | 实例属性 | `__dict__` | 存储在对象实例中的 `obj.__dict__` |
| 4 | 非数据描述符 | 类属性 | 只定义了 `__get__` 的类。如普通方法、staticmethod |
| 5 | 普通类属性 | 类变量 | 直接定义在类里的变量（且不是描述符）|
| 6 | `__getattr__` | 兜底方案 | 只有当上面的步骤都找不到属性时，才会调用它 |

```python
class DataDesc:
    def __get__(self, instance, owner):
        print("数据描述符被调用")
        return "来自数据描述符"
    def __set__(self, instance, value):
        pass

class NonDataDesc:
    def __get__(self, instance, owner):
        print("非数据描述符被调用")
        return "来自非数据描述符"

class MyClass:
    data = DataDesc()
    non_data = NonDataDesc()
    plain1 = "类属性"

    def __init__(self):
        self.data = "实例属性"
        self.non_data = "实例属性"
        self.plain2 = "实例属性"

    def __getattr__(self, name):
        print(f"__getattr__ 被调用：{name}")
        return "来自 __getattr__" # 正常应该抛出AttributeError，这里方便测试就返回字符串

    def __getattribute__(self, name):
        print(f"__getattribute__ 被调用：{name}")
        return super().__getattribute__(name)

obj = MyClass()

print("访问 data：", obj.data)
print("访问 non_data：", obj.non_data)
print("访问 plain1：", obj.plain1)
print("访问 plain2：", obj.plain2)
print("访问 missing：", obj.missing)

'''
__getattribute__ 被调用：data
数据描述符被调用
访问 data： 来自数据描述符

__getattribute__ 被调用：non_data
访问 non_data： 实例属性

__getattribute__ 被调用：plain1
访问 plain1： 类属性

__getattribute__ 被调用：plain2
访问 plain2： 实例属性

__getattribute__ 被调用：missing
__getattr__ 被调用：missing
访问 missing： 来自 __getattr__
'''
```

非数据描述符因为在实例属性查找之后类属性查找之前运行，因此可以拿来实现延迟求值。

赋值顺序 `__setattr__` -> `数据描述符` -> `__dict__`

```python
class DataDesc:
    def __set__(self, instance, value):
        print(f"数据描述符 __set__ 被调用：{value}")
    def __get__(self, instance, owner):
        return "来自数据描述符"

class MyClass:
    data = DataDesc()

    def __setattr__(self, name, value):
        print(f"__setattr__ 被调用：{name} = {value}")
        super().__setattr__(name, value)

obj = MyClass()

print("赋值 data：")
obj.data = 42

print("\n赋值 new_attr：")
obj.new_attr = 99

'''
赋值 data：
__setattr__ 被调用：data = 42
数据描述符 __set__ 被调用：42

赋值 new_attr：
__setattr__ 被调用：new_attr = 99
'''
```

`__set_name__(self, cls, name)`会在类创建之后任何实例创建之前执行，告诉描述符实例自己在哪个类里，以及自己在那个类里叫什么名字
